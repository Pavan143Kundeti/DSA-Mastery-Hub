# Python Introduction

## What is Python?

Python is a **popular programming language** created by **Guido van Rossum** and released in **1991**.

### Key Information
- **Creator**: Guido van Rossum
- **Release Year**: 1991
- **Type**: High-level, interpreted programming language
- **Philosophy**: Simple, readable, and powerful

## Python Syntax vs Other Languages

### Key Differences

#### 1. **Line Endings**
```python
# Python - Uses new lines
print("Hello World")
print("Next line")

# Other languages - Often use semicolons
// JavaScript
console.log("Hello World");
console.log("Next line");
```

#### 2. **Code Blocks & Indentation**
```python
# Python - Uses indentation (whitespace)
if True:
    print("This is indented")
    print("This too")

# Other languages - Use curly brackets
// JavaScript/Java/C++
if (true) {
    console.log("This is in brackets");
    console.log("This too");
}
```

#### 3. **Readability Focus**
```python
# Python - Reads like English
if user_age >= 18:
    print("You can vote")
else:
    print("Too young to vote")

# C++ - More symbols and syntax
if (userAge >= 18) {
    cout << "You can vote" << endl;
} else {
    cout << "Too young to vote" << endl;
}
```
